# Fundamentals Setup Guide

## Unity Installation
1. Download Unity Hub from the official Unity website.
2. Install Unity Hub and open it.
3. Create a Unity ID or log in if you already have one.
4. In the Unity Hub, navigate to the "Installs" section and click on "Add".
5. Select the latest version of Unity and follow the installation instructions.

## Creating a 2D Project
1. Open Unity Hub and go to the "Projects" tab.
2. Click on "New Project".
3. Choose the 2D template.
4. Name your project and select the desired location.
5. Click "Create" to start your new 2D project.

## Scene Setup
1. Open the new 2D project in Unity.
2. In the "Hierarchy", right-click and create a new 2D object, such as a Sprite or Tilemap.
3. Use the "Inspector" window to adjust properties such as position, scale, and rotation.
4. Save the scene by navigating to `File > Save As` and name it appropriately (e.g., MainScene).

## Camera Configuration
1. Select the "Main Camera" object in the Hierarchy.
2. In the Inspector, set the projection to "Orthographic" for 2D games.
3. Adjust the size of the orthographic camera to fit your 2D scene layout.
4. Set the background color to match your design requirements.

## Import Settings for Sprites
1. Import sprites by dragging them into the "Assets" folder from your file explorer.
2. Select each sprite in the Assets folder and adjust import settings in the Inspector:
   - Set Texture Type to "Sprite (2D and UI)"
   - Click "Apply" to save changes.

## Tilemap Basics
1. Create a Tilemap by right-clicking in the Hierarchy and selecting `2D Object > Tilemap > Rectangle`.
2. Open the Tile Palette window by selecting `Window > 2D > Tile Palette`.
3. Create a new palette and drag your tiles into it.
4. Use the Brush tool to paint tiles onto the Tilemap in your scene.

With these steps, you are now ready to start developing your 2D game project in Unity!
