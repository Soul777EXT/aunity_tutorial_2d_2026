# Unity 2D Top-Down Tutorial (Complete Series)

This folder contains a comprehensive, part-by-part tutorial series for creating professional 2D top-down games in Unity using C#.

## Structure
- PART 01: Fundamentals & Setup
- PART 02: Player Controller (Detailed)
- PART 03: Camera Systems
- PART 04: Enemy Systems
- PART 05: Combat System
- PART 06: UI System
- PART 07: Inventory & Items
- PART 08: Dialogue & NPC System
- PART 09: Audio System
- PART 10: Particle Effects & VFX
- PART 11: Level Design & Tilemap
- PART 12: Save System
- PART 13: Performance Optimization
- PART 14: Advanced Topics
- PART 15: Complete Example Projects
- PART 16: Troubleshooting & Best Practices

Each part includes explanatory text, code samples, Unity project settings, step-by-step workflows, diagrams (where applicable), and references to example scenes and scripts in Assets/Scripts/Examples.

## How to use
- Open the Example project scenes in the ExampleProjects folder (when provided).
- Read parts in order; each part builds on the previous ones.
- Code examples are placed in Assets/Scripts/Examples and are ready to use or adapt.
